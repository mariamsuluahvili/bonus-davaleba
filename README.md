Cinema Seat Booking App

A simple cinema ticket booking web application built with **HTML, CSS, and Vanilla JavaScript**.

Project Description
This project allows users to:
- Browse movies
- Choose available showtimes
- Select cinema seats
- Simulate payment
- Receive a booking confirmation

All data handling is done on the client side using **localStorage**.

---

Features Checklist
Home page with movie rendering from JSON
Movie details page with showtimes
Seat selection (8x10 hall)
Seat states: Available / Selected / Taken
Maximum 6 seats per booking
Checkout page with order summary
Payment form validation
Booking confirmation page
localStorage used for taken seats
-Responsive layout

- HTML5
- CSS3
- JavaScript (Vanilla)
